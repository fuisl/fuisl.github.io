## SS-OCT Image Analysis

### Overview
Collaborated with Team Ultrabot on a deep-learning pipeline for OCT de-noising, super-resolution, and classification. Achieved **First Runner-up** at IEEE Video and Image Processing Cup (ICIP 2024) in Abu Dhabi.

### Key Achievements
* **First Runner-up** at IEEE VIP Cup (ICIP 2024)
* Developed comprehensive pipeline for OCT image processing
* Researched and fine-tuned Vision Transformer (ViT) models for multiple subtasks

### Project Timeline
June 2024

### Key Features
* Deep-learning based OCT image de-noising
* Super-resolution techniques for enhanced image quality
* Classification models for diagnostic support
* Vision Transformer (ViT) fine-tuning and optimization

### Technologies Used
* Vision Transformers (ViT)
* Deep Learning
* Computer Vision
* OCT Image Processing

### Recognition
Official results: [2024 IEEE VIP Cup Winners](https://2024.ieeeicip.org/ieee-video-and-image-processing-cup/)

### Impact
This project demonstrates state-of-the-art techniques in medical image processing, with potential applications in ophthalmology and other medical imaging domains.
