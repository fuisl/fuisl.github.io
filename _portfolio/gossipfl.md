## GossipFL: Decentralized Federated Learning

### Overview
Developed a Proof-of-Concept for a peer-to-peer GossipFL framework with RAFT consensus algorithm to train AI models in a decentralized manner.

### Key Features
* Implemented peer-to-peer federated learning architecture
* Integrated RAFT consensus algorithm for distributed model training
* Designed decentralized communication protocols for model updates
* Demonstrated scalability and fault tolerance in distributed learning environments

### Technologies Used
* Federated Learning
* RAFT Consensus Algorithm
* Distributed Systems
* Peer-to-peer networking
